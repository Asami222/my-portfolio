# PORTFOLIO
### プロジェクトについて
<!-- プロジェクトについて -->
自身の今までに制作したWebサイトやWebアプリケーション、主に制作時に関するブログ、制作に使用したイラストレーション、お問い合わせページがあるポートフォリオサイトです。<br/>
デザインに、UIコンポーネントライブラリのMUIを取り入れました。理由は、ポートフォリオなので見やすさを特に重視したかったからと、MUIを1度使用してみたかったというのがあります。デザインに統一感が出るのでシンプルな構成にしたいポートフォリオには適していると感じました。MUIを使用することでUI/UXの勉強にもなります。私はFigmaでのデザインの時点でMaterial UI for Figmaを使用しデザインしましたが、コンポーネントの作り方などとても学びがありました。
